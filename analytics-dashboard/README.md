# Analytics Dashboard

Single-screen data dashboard inspired by enterprise BI interfaces. Dark theme, glowing charts, dense layout.

## Features

- **Hero gauge** — radial performance indicator (73%)
- **KPI cards** — Active Users, Conversion, Revenue, NPS
- **Mini gauges** — Capacity / server load
- **Bar charts** — Channel breakdown, alerts
- **Line chart** — 7-day trend with gradient fill
- **Quick stats** — Orders, Sessions, Uptime, Rating

## Tech

- Single HTML, vanilla JS
- CSS variables, SVG for charts
- Glow effects: `drop-shadow`, `box-shadow`
- Works on any static hosting (reg.ru)

## Deploy

Upload `index.html` to your web root.
