# Chromatic Year

> Each day is a color. Your year becomes a painting.

A single-screen personal journal where every day of the year is a small colored tile — the color reflects your self-rated **energy**, **mood**, and **focus**. By December the grid becomes a vivid mosaic: cold blues for hard days, warm golds for great ones.

Inspired by GitHub contribution graphs and pixel art calendars, but built around meaning.

## Features

- **Full-year mosaic grid** — 365 clickable tiles, colored by your entries
- **3-metric system** — Energy (⚡), Mood (😊), Focus (🎯) each rated 1–5
- **Color science** — hue (mood), saturation (energy), lightness (combined avg)
- **Day modal** — click any past day to rate + add a note
- **Stats chips** — current streak, % logged, averages per metric
- **Monthly activity bar chart** — see which months you logged most
- **30-day trend lines** — all 3 metrics over the past 30 days
- **Export / Import JSON** — backup your whole year
- **RU / EN** — full interface translation, locale-aware dates
- **localStorage** — no server, no account, fully private

## Deploy

Static single file — works on any host including reg.ru. Just upload `index.html`.
