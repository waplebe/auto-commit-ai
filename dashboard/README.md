# Dashboard — Прогресс дня

Комплексный одностраничный дашборд на Next.js: прогресс дня, фазы, стоимость времени, визуализация.

## Стиль

Тёмная тема, неоморфизм, оранжево-розовые градиенты, силуэты гор, RU/EN.

## Деплой на reg.ru

```bash
npm run build
```

После сборки папка `out/` содержит статику — залей её содержимое в `www` на хостинге (HTML, _next, и т.д.).

## Локально

```bash
npm run dev
```
