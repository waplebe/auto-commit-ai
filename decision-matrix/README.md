# Decision Matrix

**Матрица решений** — взвешенная оценка вариантов по критериям. Для сложных выборов: работа, переезд, покупка.

## Как устроено

- **Варианты** (строки) — что сравниваем: Job A, Job B, Остаться, Уехать
- **Критерии** (столбцы) — по чему оцениваем: Зарплата, Рост, Локация, Work-life
- **Веса** (1–10) — важность критерия
- **Оценки** (1–10) — как вариант выполняет критерий

**Итог:** взвешенная сумма → рейтинг и визуализация.

## Функции

- Добавление вариантов и критериев (до 10)
- Редактирование имён прямо в таблице
- Радар-диаграмма топ-3
- Бар-сравнение
- Сохранение в localStorage
- RU/EN

## Деплой

Один `index.html` — работает на любом статическом хостинге (reg.ru, GitHub Pages, Vercel).
