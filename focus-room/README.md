# Focus Room

Комната для фокуса: ambient-звуки + Pomodoro-таймер + логирование сессий.

## Возможности

- **5 видов звука** — белый шум, розовый, коричневый, дождь, ручей (синтез через Web Audio API, без файлов)
- **Пресеты** — Дождь, Кафе, Фокус, Выкл
- **Pomodoro-таймер** — 25 мин фокус → 5 мин перерыв
- **Задача** — фиксируешь, над чем работаешь
- **Полноэкранный режим** — минималистичный экран во время сессии
- **Статистика** — минуты за сегодня, количество сессий, стрик дней
- **История** — список завершённых сессий за день

## Использование

Открой `index.html` в браузере. Данные хранятся в `localStorage`.

## Деплой

Один HTML-файл — работает на любом статическом хостинге (reg.ru, GitHub Pages, Vercel).

**Примечание:** Web Audio API требует пользовательского взаимодействия (клик) перед воспроизведением. Первый клик по звуку или кнопке «Старт» активирует аудио.
