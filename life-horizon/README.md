# Life Horizon

**Идея:** Время — конечный ресурс. Дашборд показывает прожитые и оставшиеся недели, помогает сосредоточиться на одном деле и замечать хорошее.

**Функции:**
- Линия жизни — визуальный прогресс (прожито / осталось)
- Прожито недель — с момента рождения
- Осталось недель — при ожидании 80 лет
- Одна вещь на сегодня — фокус по «The One Thing»
- Что хорошего сегодня — практика благодарности

**Один файл.** Работает везде. Данные в localStorage.
