# Стоимость твоей минуты

Одностраничный веб‑сервис: вводишь зарплату → видишь стоимость каждой секунды, минуты, часа. Плюс «живой» счётчик: сколько рублей «сгорает», пока смотришь на экран.

## Деплой на reg.ru

1. Зайди в панель хостинга (ISPmanager)
2. Через FTP или файловый менеджер залей `index.html` в `www` (или `public_html`)
3. Готово. Никакого Node, PHP, баз данных — только статика.

## Файлы

- `index.html` — всё в одном файле (HTML + CSS + JS)
- Шрифты подгружаются с Google Fonts (работает на любом хостинге)

## Тредс

Сервис сделан так, чтобы скриншот выглядел цельно и профессионально — один экран, один фокус, понятный результат.
