# Nexus — Command Center

Single-screen productivity dashboard. Dense, complex, beautiful.

## Features

- **Clock** — Live time, date, greeting
- **Day Progress** — Circular % of day passed, phase (Night/Morning/Day/Evening), 24-hour bar
- **Focus Gauge** — Score 0–100% based on today's focus sessions
- **Stats** — Minutes today, streak days
- **Week Chart** — Last 7 days bar visualization
- **Moment** — Rotating micro-break hints (breathe, stand, water...)
- **Now/Task** — Current task input, quick 25/5 min timer buttons
- **Sessions** — Today's completed sessions list
- **Sounds** — White, brown, rain sliders (Web Audio)
- **Timer Overlay** — Full-screen when running focus

## Design

Dark theme, orange-pink gradients, neumorphic cards, mountain silhouettes. Responsive grid.

## Deploy

Single `index.html` — works on any static hosting (reg.ru, Vercel, etc).
