# Synapse Desk

Одностраничный **персональный командный центр**: матрица Эйзенхауэра с drag-and-drop, Pomodoro с недельной статистикой, трекер привычек на 7 дней, быстрый журнал, экспорт/импорт JSON. Всё в браузере (`localStorage`), без бэкенда — удобно для статического хостинга (в т.ч. reg.ru).

## Возможности

- **Эйзенхауэр** — задачи по квадрантам, перетаскивание между зонами, чекбоксы «готово»
- **Орбита дня** — кольцевой прогресс суток, фаза (ночь / утро / день / вечер)
- **Pomodoro** — 25 / 5, пауза, пропуск фазы; минуты фокуса пишутся в статистику по дням
- **График за 7 дней** — накопленные минуты Pomodoro + счётчик записей журнала за неделю
- **Привычки** — до 7 дней назад, галочки по дням
- **Журнал** — короткие записи с датой
- **RU / EN**, резервная копия JSON

## Запуск

Откройте `index.html` в браузере или положите папку на любой статический хостинг.
