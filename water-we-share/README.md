# Water We Share (Вода, которой мы делимся)

Одностраничное веб-приложение про **глобальный дефицит воды** и личное бытовое потребление.

## Зачем

- По данным ООН, **сотни миллионов** людей не имеют безопасного доступа к питьевой воде; **миллиарды** — к безопасному санитарному обслуживанию.
- Сервис не «стыдит», а **показывает порядок величин**: душ, смывы, стирка, посуда, питьё — и сравнение с грубыми ориентирами (ВОЗ, средние по миру, зона сильного водного стресса).

## Как пользоваться

Откройте `index.html` в браузере. Подвиньте слайдеры — пересчёт мгновенный. Можно **сохранить сценарий** в `localStorage`. Есть **RU / EN**.

## Деплой

Статический файл — подходит для любого хостинга (в т.ч. reg.ru): загрузите папку или один `index.html`.

## Оговорка

Цифры **упрощённые** (типичные расходы лейки, смыва, машины), для осознанности, не для инженерного аудита.
